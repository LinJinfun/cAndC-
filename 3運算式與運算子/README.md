# 資料型別和變數
## 運算式

```
運算子: +,-,*,/
運算元: 1+2的1和2
單元運算子:-24的-
二元運算子:1+2的+
三元運算子: true ? a : b
```


算數運算子|意義|
-----|----|
+| 加 |
-| 減 |
*|乘|
/|除|
++|遞增|
--|遞減|





```
	int a, b;
	cout << "請輸入被除數(整數):";
	cin >> a;
	cout << "請輸入除數(整數,不能為0):";
	cin >> b;
	cout << "商:" << a/b << ",餘數:" << a%b << "\n";

	//前置遞增
	int x = 1;
	int y = ++x;
	//x = x + 1;
	//y = x;
	cout << "x = " << x << " y = "  << y << endl;

	//後置遞增
	int m = 1;
	int n = m++;
	//n = m;
	//m = m + 1
	cout << "m = " << m << " n = "  << n << endl;
```
