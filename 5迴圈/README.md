## 巢狀迴圈
	//利用2層迴圈列印「井」字，將其排列成直角三角形
		for(int i=1; i<=5; i++){
			for(int j=1; j<=i; j++){
				cout << "#";
			}
			cout << "\n";
		}

## 不固定次數的迴圈
```
while(條件判斷)
{
程式敘述;
........
}
```

```
while迴圈要特別注意，必須設定條件判斷的中止條件，以便可以停止迴圈的執行，否則會入無窮迴圈的窘境
//無限迴圈
int n=1, sum;
while(n<10){
	sum += n;
}
```

	//小明想要存錢買一輛機車,機車每輛30000元，他將每月存的錢輸入，當存款足夠買機車時，就顯示提示訊告知。
	#include <iostream>
	using namespace std;

	int main() {
		int deposit = 0, count = 0, n;
		while(deposit < 30000){
			count++;
			cout << "請輸入第" << count << "個月份的存款:";
			cin >> n;
			deposit += n;
		}
		cout << "存了" << count << "個月的總存款為:" << deposit << "元。\n已經足夠購買機車。";
		return 0;
	}


```
while迴圈取代for迴圈
for(int i=0; i<=5; i++)
{
	cout << "這是第" << i << "次迴圈";
}

//使用while迴圈達到相同結果
int i=1;
while(i<=5){
	cout << "這是第" << i << "次迴圈";
}
```

			cout << "請輸入密碼:";
